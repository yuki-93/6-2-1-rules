# 6-2-1-rules
6-2-1 rules generator

Generate 6-2-1 rules. Data source: https://pad.medialepfade.net/6-2-1-Generator?both

Inspired by CCC-events 6-2-1 Rule: 6 hours of sleep, 2 meals, 1 shower - repeat daily.

## Prod
```sh 
yarn install 
yarn build
# serve dist/ folder
```

## Development
```sh
yarn install
yarn dev
# open browser 
```
